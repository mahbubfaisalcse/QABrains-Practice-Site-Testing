# 🧪 QA Portfolio: QA Brains Practice Platform Testing


## 📖 Project Overview

This repository showcases a complete **Manual Software Quality Assurance (SQA)** project conducted on the **QA Brains Practice Platform**.

The project demonstrates the complete Software Testing Life Cycle (STLC), including requirement analysis, test planning, test case design, execution, defect reporting, and test summary documentation.

The objective was to validate the application's functionality, usability, UI consistency, and user experience while identifying defects and documenting them using industry-standard QA practices.

---

# 🎯 Project Objectives

- Verify core application functionality.
- Validate business requirements through comprehensive test cases.
- Identify functional, UI/UX, usability, and validation issues.
- Report defects with clear reproduction steps and evidence.
- Produce professional QA documentation suitable for real-world software projects.
